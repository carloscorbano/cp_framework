My Project is a C++ application focused on high-performance rendering using Vulkan. Its goal is to provide an easy-to-use framework with clean APIs, expressive abstractions, and strong RAII guarantees.

  • Strong separation of responsibilities

ðŸ§© Modules Overview

The project is divided into the following main modules:

  • Core: basic utilities, memory, filesystem, logging
  • Graphics: Vulkan wrapper, pipelines, command buffers
  • Events: global dispatcher, strongly-typed event system
  • Math: vectors, matrices, transforms

Build Instructions

  1. Install dependencies (Vulkan SDK, GLFW, etc.)
  2. Configure the project:
    cmake -B build -DCMAKE_BUILD_TYPE=Release
  3. Compile:
    cmake --build build
  4. Run the executable inside the /bin folder.
