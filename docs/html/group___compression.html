std::vector< uint8_t > cp::compression::CompressData ( std::span< const uint8_t > data,
int level = 1 )

Compresses the given data using zlib and embeds the original size.

The resulting buffer format is:

[8 bytes: original_size (uint64_t)] [compressed data...]
Parameters
dataByte span to be compressed.
levelCompression level (1 = fastest, 9 = best compression, default = 1 / Z_BEST_SPEED).
Returns
std::vector<uint8_t> A buffer containing the 8-byte header followed by the compressed payload.
Note
Returns an empty vector in case of compression failure.

std::vector< uint8_t > cp::compression::UncompressData ( std::span< const uint8_t > compressedData,
uint64_t maxAllowedSize = 4ULL * 1024 * 1024 * 1024 )

Decompresses data produced by CompressData().

The function reads the original uncompressed size stored in the 8-byte header and reconstructs the original data.

Parameters
compressedDataSpan containing the 8-byte header + compressed payload.
maxAllowedSizeMaximum allowed size for decompression (default: 4 GiB), used to prevent decompression-bomb attacks.
Returns
std::vector<uint8_t> A vector containing the decompressed data.
Note
Returns an empty vector if:
  • the header is invalid
  • the input data is corrupted
  • decompression fails
  • declared size exceeds maxAllowedSize
