Detailed Description

Implements the MD5 hashing algorithm.

This class provides a full implementation of the MD5 algorithm, including incremental updates, finalization, and hexadecimal output.

Typical usage:

MD5 md5;
md5.update("hello");
md5.finalize();
std::string hash = md5.hexdigest();
